# Complete Practical Exploitation Guide: X11 Security (Crest CRT Level)

## 1. X11 Fundamentals & Attack Surface (From Scratch)

**What is X11?**
- **X Window System** (TCP/UDP port 6000+N, default 6000)
- Client-server architecture: **Display = Server**, Apps = Clients
- **Network-transparent** by default (huge attack surface)
- Magic cookie auth (MIT-MAGIC-COOKIE-1) - easily crackable
- **NO encryption**, keystrokes/pixels transmitted in plaintext

**Security Attributes:**
```
- xhost +           = NO AUTH (anyone connects)
- xhost +hostname   = Trust specific hosts  
- xhost -            = Host-based ACL only
- ~/.Xauthority     = User-based cookie auth
- $DISPLAY=:0        = Local only (unix socket)
- $DISPLAY=192.168.1.100:0 = Remote listening
```

**Attack Primitives:**
1. **Connection hijacking** (xhost +)
2. **Keystroke injection**
3. **Screenshot capture**
4. **Command injection** (xterm takeover)
5. **Cookie cracking** (MIT-MAGIC-COOKIE)
6. **Privilege escalation** via trusted apps
